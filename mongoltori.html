BRACU Mongol-Tori

BRACU Mongol-Tori was a pioneering Mars rover project developed by BRAC University's robotics team. As a key team member, I contributed to designing and implementing various systems for this rover that competed in international competitions.

The project aimed to create a fully functional Mars rover prototype capable of traversing difficult terrain, collecting samples, and performing scientific analysis - similar to rovers deployed on actual Mars missions.

Key Features

Custom Chassis

Designed for optimal performance on rough terrain with advanced rocker-bogie suspension system inspired by NASA's Mars rovers.

Robotic Arm

5-DOF robotic arm with specialized end effector for soil sampling and precision manipulation of objects in the Martian terrain simulations.

Real-time Control

Custom control station with real-time video transmission and telemetry feedback enabling precise remote operations during competition challenges.

Sensor Systems

Integrated sensor array for environmental data collection, obstacle detection, and autonomous navigation using computer vision algorithms.

Power Management

Efficient power distribution system with solar panel integration, providing extended operational time during competition challenges.

    2021
